# common v2.0.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver `github.com/xpfo-go/common/v2` v2.0.0 with safer APIs, deterministic tests, and GitHub-standard automation/docs.

**Architecture:** Keep package structure stable while introducing explicit error returns for failure-prone APIs. Replace runtime file dependency with embedded assets, split integration tests by environment gating, and enforce quality via CI workflows.

**Tech Stack:** Go 1.22+, go:embed, go test/vet, golangci-lint, GitHub Actions, markdown governance docs.
